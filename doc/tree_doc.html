首先引入jquery库，然后引入tree.js、tree.css，如下： 

<script type="text/javascript" src="js/jquery/jquery-1.7.2.min.js"></script>

<script type="text/javascript" src="js/tree.js"></script>
<link rel="stylesheet" href="css/blue/tree.css"/>

在页面加载完成后初始化tree插件，我们以ajax结合json方式实现 

$("#tree3").tree({
	url: "ajax/ajax_tree.jsp",
	onClick: function(node) {
		console.log(node);
		alert(node.text);
	}
});

远程地址需要返回json格式的树节点数据，类似： 

[
	{
		id: 1, 
		text: "", 
		open: true|false, 
		type: "folder", 
		children: [], 
		url: ""
	}, 
	{
		id: 2, 
		text: "", 
		url: ""
	}
]

页面代码 

<div id="tree3"></div>

tree插件有以下初始化选项

选项 类型 作用
data Object JSON格式数据，形如：
[
    {
        id: 1,
        text: "",
        open: true|false,
        type: "folder",
        children: [
            {
                id: 3,
                text: "",
                url: ""
            }
        ],
        url: ""
    },
    {
        id: 2,
        text: "",
        url: ""
    }
]
url string 获取节点数据的远程地址
onClick function 节点点击函数，插件会把被点击的节点对象传给这个函数

tree插件有以下函数

函数名 参数 功能 返回值 示例
open folderNodeId 打开指定节点，前提是该节点已经存在且是一个folder 
$("#btn1").click(function() {
	$("#tree3").tree("open", 9);
});
close folderNodeId 关闭指定节点，前提是该节点已经存在且是一个folder 
$("#btn3").click(function() {
	$("#tree3").tree("close", 9);
});
openAll 打开全部folder节点 
$("#btn5").click(function() {
	$("#tree3").tree("openAll");
});
closeAll 关闭全部folder节点 
$("#btn7").click(function() {
	$("#tree3").tree("closeAll");
});
reload folderNodeId 重新加载指定folder节点 
$("#btn9").click(function() {
	$("#tree3").tree("reload", 9);
});
