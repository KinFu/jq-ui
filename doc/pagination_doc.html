首先引入jquery库，然后引入pagination.js、pagination.css，如下： 

<script type="text/javascript" src="js/jquery/jquery-1.7.2.min.js"></script>

<script type="text/javascript" src="js/pagination.js"></script>
<link rel="stylesheet" href="css/blue/pagination.css"/>

在页面加载完成后初始化pagination插件 

<script>
	// 分页插件
	$("#pagination").pagination({
		size: 6,
		total: 11,
		click: function(curr, s) {
			return 11;
		}
	});
</script>

页面代码 

<!-- 分页插件 -->
<div id="pagination"></div>

pagination插件有以下初始化选项

选项 类型 作用
pageNum int 第几页，默认1
size int 显示的页码个数，默认10
total int 总页数
click function 页码链接点击时执行的函数，您可以在这个函数里面异步加载表格数据，该函数需要返回总页数

pagination插件有以下函数

函数名 参数 功能 返回值 示例
getCurrentPage 获取当前页码 int类型的当前页码 
$("#pagination").pagination(
	"getCurrentPage"
);
toPage object，如：{"current": 1, "total": 11} 跳转至指定页码 
$("#pagination").pagination(
	"toPage", {"current": 1, "total": 11}
);
getSize 获取显示的页码个数 int类型 
$("#pagination").pagination(
	"getSize"
);
