首先引入jquery库，然后引入accordion.js、accordion.css，如下： 

<script type="text/javascript" src="js/jquery/jquery-1.7.2.min.js" ><"/script>

<script type="text/javascript" src="js/pagination.js" ></script>
<link rel="stylesheet" href="css/pagination.css" />

在页面加载完成后初始化accordion插件 

<script>
	// 分页插件
	$("#pagination").pagination({
		"size": 6,
		"click": function(curr, s) {
			return 11;
		}
	});
	$("#pagination").pagination(
		"toPage", {"current": 1, "total": 11}
	);
</script>

页面代码 

<!-- 分页插件 -->
<div id="pagination"></div>

pagination插件有以下初始化选项

选项 类型 作用
size int 每一页的大小
click function 页码链接点击时执行的函数，您可以在这个函数里面异步加载表格数据，该函数需要返回总页数

pagination插件有以下函数

函数名 参数 功能 返回值 示例
getCurrentPage 获取当前页码 int类型的当前页码 
$("#pagination").pagination(
	"getCurrentPage"
);
toPage {"current": 1, "total": 11} 跳转至指定页码 
$("#pagination").pagination(
	"toPage", {"current": 1, "total": 11}
);
getSize 获取每一页的大小 int类型 
$("#pagination").pagination(
	"getSize"
);
