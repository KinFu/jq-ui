首先引入jquery库，然后引入datatable.js、datatable.css，如下： 

<script type="text/javascript" src="js/jquery/jquery-1.7.2.min.js"></script>

<script type="text/javascript" src="js/datatable.js"></script>
<link rel="stylesheet" href="css/blue/datatable.css"/>

在页面加载完成后初始化datatable插件，以下代码只是最初步的初始化datatable插件样式，至于详细的交互过程（如：表格数据刷新、tab标签关闭打开等）还需要参考“异步分页表格插件示例” 

<script>
	$("#first-datatable").datatable({
		width: "1000",
		height: "auto",
		columns: [
			{ field: "id", columnName: "编号", css: { "text-align": "center" } },
			{ field: "username", columnName: "用户名" },
			{ field: "age", columnName: "年龄" },
			{ field: "phone", columnName: "联系电话", css: { "text-align": "center" } },
			{ field: "email", columnName: "邮箱" },
			{ field: "description", columnName: "自我介绍" }
		],
		url: "/jq-ui/ajax/admin_json.jsp", // 获取数据的远程地址，返回数据格式如：{ totalPage: 1, rows: [] }
		pageNum: 1,			// 显示第几页数据，默认1
		pageSize: 15,		// 每页数据数量，默认10
		pagination: true,	// 是否启用分页组件，默认启用
		showCheckbox: true
	});
</script>

datatable插件有以下初始化选项

选项 类型 作用
width int/百分比 表格的宽，默认100%
height int 表格的高，默认auto
columns Array 字段信息，默认是[]。如：[ { field: "", columnName: "", css: { "text-align": "center" }, format: function(v) {return v;} }, { field: "", columnName: "", css: { "text-align": "center" } }, ... ]
url string 获取数据的远程地址，返回数据格式如：{ totalPage: 1, rows: [] }
pageNum int 显示第几页数据，默认1
pageSize int 每页数据数量，默认10
pagination boolean 是否启用分页组件，默认启用
showCheckbox boolean 行首是否显示复选框，默认不显示

datatable插件有以下函数

函数名 参数 功能 返回值 示例
reload 重新加载当前页数据 
$("#first-datatable").datatable("reload");
getSelectRows 获取选择的行记录 Array 
var rows = $("#first-datatable").datatable("getSelectRows");
