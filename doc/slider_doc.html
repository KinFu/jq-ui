首先引入jquery库，然后引入jquery.slider.js、slider.css，如下： 

<script type="text/javascript" src="js/jquery/jquery-1.7.2.min.js"></script>

<script type="text/javascript" src="js/slider/jquery.slider.js"></script>
<link rel="stylesheet" href="js/slider/slider.css"/>

在页面加载完成后初始化slider插件 

<script>
	$("#banner1").slider({
		timeInterval: 2000, // 轮播时间间隔，默认4s
		controlNav: true,   // 圆点导航按钮
		directionNav: true  // 上一个和下一个按钮
	});
	$("#banner").slider({
		timeInterval: 2000,
		controlNav: true,
		directionNav: true
	});
</script>

slider插件有以下初始化选项

选项 类型 作用
directionNav boolean 是否显示上一个和下一个按钮，默认true
controlNav boolean 是否显示圆点导航按钮，默认true
width int 宽度，默认使用图片的宽度
height int 高度，默认使用图片的高度
timeInterval int 轮播时间间隔，默认4000，即4s
