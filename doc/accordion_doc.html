首先引入jquery库，然后引入accordion.js、accordion.css，如下： 

<script type="text/javascript" src="js/jquery/jquery-1.7.2.min.js"></script>

<script type="text/javascript" src="js/accordion.js"></script>
<link rel="stylesheet" href="css/blue/accordion.css"/>

在页面加载完成后初始化accordion插件 

<script>
	$(function() {
		$("#accordion1").accordion();
	});
</script>

accordion插件有以下函数

函数名 参数 功能 返回值 示例
select accordionId 选中指定菜单 
$('.accordion').accordion(
	'select', 'menu2'
);
isSelected accordionId 判断指定指定菜单是否选中 true/false 
$('.accordion').accordion(
	'isSelected', 'menu2'
);
