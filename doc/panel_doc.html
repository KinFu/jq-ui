首先引入jquery库，然后引入panel.js、panel.css，如下： 

<script type="text/javascript" src="js/jquery/jquery-1.7.2.min.js"></script>

<script type="text/javascript" src="js/panel.js"></script>
<link rel="stylesheet" href="css/blue/panel.css"/>

在页面加载完成后初始化panel插件 

<script>
	$("#panel1").panel({
		title: "第一个panel面板",
		width: "50%",
		height: "80%",
		url: "ajax/admin_panel.html"
	});
</script>

页面代码 

<div id="panel1"></div>
<br />

panel插件有以下初始化选项

选项 类型 作用
width int/百分比 面板div的宽，默认不设置
height int 面板div的高，默认不设置
title string 如果设置为空串、null或者undefined就不显示
url string 获取数据的远程地址

panel插件有以下函数

函数名 参数 功能 返回值 示例
reload [url] 重新加载数据 无 
function refresh() {
	$("#panel1").panel("reload");
}
options 无 获取面板的配置信息 Object 
var options = $("#panel").panel("options");
